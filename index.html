🕹️ ¿Cómo se juega?
Primero se escribe el nombre de quien reta 🗡️ y de quien acepta el duelo 🛡️.

De forma opcional, se pueden usar los filtros por familia y nivel para personalizar el reto, o jugar directamente al azar 🎲.

Después se roba un cromo compartido 🃏.
Ese será el ejercicio que ambos deberán realizar.

A continuación, se tiran los dados 🎲 para establecer la carga del ejercicio:
repeticiones, tiempo o ciclos.

Desde ese momento comienza la acción:
- Una persona ejecuta el ejercicio intentando cumplir:
  - el indicador de logro del cromo 💪
  - y la carga marcada por los dados 🎲
- La otra persona observa y ayuda señalando posibles errores de ejecución 👀🔍

Después se cambian los roles 🔁.

⚔️ ¿Cómo termina el duelo?
El resultado depende de:
- la carga marcada por los dados
- el indicador de logro del cromo

❌ Nadie cumple
→ nuevo cromo o finalizar partida

🗡️ Gana Retador
→ añade el cromo a su álbum

🛡️ Gana Retado
→ añade el cromo a su álbum

🤝 Empate
→ se activa el desempate

⚡ Desempate pedagógico
En caso de empate, se activa una muerte súbita pedagógica sin azar con preguntas paralelas para ambos jugadores.

Completar los 36 cromos del Álbum Atleta EFBoral 💪🧠
